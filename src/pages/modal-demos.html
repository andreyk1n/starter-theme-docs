Приклади модальних вікон і попапів

На цій сторінці ви знайдете приклади різних модальних вікон і попапів із різноманітними анімаціями появи та унікальною структурою, які працюють без використання javascript коду, хіба для автоматичного виведення можемо використовувати заготований скрипт таймера.

Базова верстка для усіх модальних вікно (на цій сторінці)

                        
<div class="modal">
    <div class="modal-content">
        <span onclick="document.location='#close'; return false;" class="close"></span>
        <h2>Модальне вікно</h2>
        <p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Enim corrupti necessitatibus
            sint quas error soluta voluptatibus? Delectus accusamus necessitatibus omnis iure cum
            architecto. Nesciunt aspernatur tempora minima sint quae eligendi.</p>
    </div>
    <span onclick="document.location='#close'; return false;" class="close-overlay"></span>
</div>
