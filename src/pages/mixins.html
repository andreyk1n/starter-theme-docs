Міксини та Функції

У цьому розділі описані SCSS міксини та функції (файл _mixins.scss), які використовуються в проекті для спрощення розробки та забезпечення адаптивності стилів.

Міксини

Міксини дозволяють повторно використовувати фрагменти CSS-коду з параметрами, що спрощує підтримку та розширення стилів.

Міксин adaptiveValue

Міксин adaptiveValue автоматично генерує адаптивні значення для вказаної CSS-властивості залежно від ширини екрану. Це корисно для таких властивостей, як шрифти, відступи або розміри елементів, забезпечуючи плавні переходи між різними роздільними здатностями. 

            
@mixin adaptiveValue($property, $maxValue, $minValue) {
    // Стандартні брекпоїнти
    $breakpoints: (
        1200px,
        992px,
        768px,
        576px,
        480px,
        360px,
        320px
    );

    // Обчислюємо кількість кроків на основі кількості breakpoints
    $steps: list.length($breakpoints);
    $step-size: math.div(($maxValue - $minValue), $steps);

    #{$property}: #{$maxValue};

    @for $i from 1 through $steps {
        $breakpoint: list.nth($breakpoints, $i);
        $current-value: $maxValue - ($step-size * $i);

        @media (max-width: $breakpoint) {
            #{$property}: $current-value;
        }
    }
}

Функції

Функції дозволяють виконувати обчислення та повертати значення, які можуть бути використані у стилях. Це допомагає зробити код більш гнучким та легким для підтримки.

Функція toRem

Функція toRem перетворює значення в пікселях у rem, що дозволяє забезпечити адаптивність тексту та елементів відносно кореневого розміру шрифту. 

            
@function toRem($px, $current: 16) {
    @return ($px / $current) * 1rem;
}

Функція toEm

Функція toEm перетворює значення в пікселях у em, що дозволяє більш гнучко керувати розмірами елементів у відношенні до їх батьківських елементів. 

            
@function toEm($px, $current: 16) {
    @return ($px / $current) * 1em;
}
