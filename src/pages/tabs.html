Таби

У цьому розділі описано принцип роботи табів, їх функції, стилі та приклади використання.

Приклад використання

Ось приклад структури HTML для табів:

            
        <div class="tabs">
            <div class="tabs__buttons">
                <button class="tabs__button tabs__button--active">Tab 1</button>
                <button class="tabs__button">Tab 2</button>
                <button class="tabs__button">Tab 3</button>
            </div>
            
            <div class="tabs__content tabs__content--active">Content for Tab 1</div>
            <div class="tabs__content">Content for Tab 2</div>
            <div class="tabs__content">Content for Tab 3</div>
        </div>

Клас tabs необхідний для контейнера з усіма табами. Клас tabs__buttons містить кнопки для кожного табу і відповідає за їх розташування в ряд, завдяки використанню display: flex. Клас tabs__button відповідає за кожну кнопку табу, з додатковим модифікатором tabs__button--active, який активує відповідний таб (змінює його стиль). Клас tabs__content містить контент для кожного табу. Клас tabs__content--active застосовується до контенту активного табу, що дозволяє відображати відповідний вміст.

Приклад стилізації

            
              .tabs {
                  display: flex;
                  flex-direction: column;
                  align-items: flex-start;
                  max-width: 500px;
              }
          
              .tabs__buttons {
                  display: flex;
              }
          
              .tabs__button {
                  padding: 8px 16px;
                  border: 1px solid #333;
                  background-color: transparent;
                  cursor: pointer;
                  margin-right: 4px;
              }
          
              .tabs__button--active {
                  background-color: #333;
                  color: #fff;
              }
          
              .tabs__content {
                  display: none;
                  padding: 16px;
                  border: 1px solid #ddd;
                  margin-top: 8px;
                  width: 100%;
              }
          
              .tabs__content--active {
                  display: block;
              }
