Таби

У цьому розділі описано принцип роботи табів, їх функції, стилі та приклади використання.

Приклад використання

Ось приклад структури HTML для табів:

            
        <div class="tabs">
            <div class="tabs__buttons">
                <button class="tabs__button tabs__button--active">Tab 1</button>
                <button class="tabs__button">Tab 2</button>
                <button class="tabs__button">Tab 3</button>
            </div>
            
            <div class="tabs__content tabs__content--active">Content for Tab 1</div>
            <div class="tabs__content">Content for Tab 2</div>
            <div class="tabs__content">Content for Tab 3</div>
        </div>

Приклад стилізації

            
        .tabs {
            display: flex;
            flex-direction: column;
            align-items: flex-start;
            max-width: 500px;
        
            &__buttons {
                display: flex;
            }
        
            &__button {
                padding: 8px 16px;
                border: 1px solid #333;
                background-color: transparent;
                cursor: pointer;
                margin-right: 4px;
        
                &--active {
                    background-color: #333;
                    color: #fff;
                }
            }
        
            &__content {
                display: none;
                padding: 16px;
                border: 1px solid #ddd;
                margin-top: 8px;
                width: 100%;
        
                &--active {
                    display: block;
                }
            }
        }
