Слайдер на ванільному JavaScript

Цей слайдер працює за принципом горизонтального прокручування контейнера зі слайдами. Вся група слайдів знаходиться всередині елемента .slider__wrapper, який має display: flex, завдяки чому слайди розташовуються в один ряд. При натисканні на кнопки «←» або «→» змінюється змінна currentIndex, яка відповідає за те, з якого слайда починати показ. Після цього функція updateSlider() змінює властивість transform: translateX() у контейнера, зсуваючи його на необхідну кількість пікселів.

Кількість одночасно видимих слайдів залежить від ширини вікна браузера і визначається функцією getSlidesToShow(). Якщо ширина екрана менша або дорівнює 600px, відображається один слайд. При ширині до 900px — два слайди. Якщо ширина більша за 900px, то на екрані видно три слайди. Це дозволяє адаптувати слайдер під різні пристрої.

Крім кнопок, слайдер підтримує свайп-жести на мобільних пристроях. При торканні екрану фіксується початкова координата пальця (startX), а при відпусканні — кінцева (endX). Якщо різниця між ними більша за 50 пікселів, визначається напрям руху, і викликається відповідна функція перемикання.

Слайдер має циклічну поведінку: коли користувач доходить до останнього слайда і натискає «→», він повертається на початок, і навпаки — при натисканні «←» на першому слайді переходить до останнього.

Необхідний JavaScript

Скрипт починає з вибору основних елементів DOM: контейнера слайдера, обгортки з усіма слайдами, масиву слайдів та кнопок керування. Змінна currentIndex зберігає поточну позицію показу, а slidesToShow визначає кількість видимих слайдів. Функція getSlidesToShow() повертає значення залежно від ширини вікна, а updateSlider() розраховує ширину одного слайда і зсуває контейнер на потрібну відстань.

Функції nextSlide() та prevSlide() змінюють currentIndex і викликають оновлення відображення. При зміні розміру вікна виконується перерахунок slidesToShow і оновлення позиції слайдів, щоб вони завжди відображались коректно.

Для підтримки сенсорних пристроїв додані обробники touchstart та touchend, які визначають напрямок свайпу та перемикають слайди відповідно.


const slider = document.querySelector('#slider');
const wrapper = slider.querySelector('.slider__wrapper');
const slides = slider.querySelectorAll('.slider__slide');
const prevBtn = slider.querySelector('#prev');
const nextBtn = slider.querySelector('#next');

let currentIndex = 0;
let slidesToShow = getSlidesToShow();

function getSlidesToShow() {
  if (window.innerWidth <= 600) return 1;
  if (window.innerWidth <= 900) return 2;
  return 3;
}

function updateSlider() {
  const slideWidth = slider.offsetWidth / slidesToShow;
  wrapper.style.transform = `translateX(${-currentIndex * slideWidth}px)`;
}

function nextSlide() {
  if (currentIndex < slides.length - slidesToShow) {
    currentIndex++;
  } else {
    currentIndex = 0;
  }
  updateSlider();
}

function prevSlide() {
  if (currentIndex > 0) {
    currentIndex--;
  } else {
    currentIndex = slides.length - slidesToShow;
  }
  updateSlider();
}

nextBtn.addEventListener('click', nextSlide);
prevBtn.addEventListener('click', prevSlide);

window.addEventListener('resize', () => {
  slidesToShow = getSlidesToShow();
  updateSlider();
});

// Свайп
let startX = 0;
let endX = 0;

wrapper.addEventListener('touchstart', e => {
  startX = e.touches[0].clientX;
});

wrapper.addEventListener('touchend', e => {
  endX = e.changedTouches[0].clientX;
  if (startX - endX > 50) {
    nextSlide();
  } else if (endX - startX > 50) {
    prevSlide();
  }
});

updateSlider();

Необхідні стилі

Слайдер має overflow: hidden, щоб приховувати непотрібні слайди за межами видимої області. Обгортка .slider__wrapper використовує display: flex і плавний перехід transition для анімації зміщення. Слайди мають фіксовану ширину, яка змінюється через flex: 0 0 calc(100% / N) залежно від кількості видимих слайдів. Картки мають білий фон, тінь і округлені кути для приємного вигляду. Кнопки керування розташовані поверх слайдера в правому нижньому куті.

Через медіа-запити змінюється ширина слайдів, щоб на планшетах відображалося два слайди, а на телефонах — один.
