Скрипт приховування/показу заголовка при прокручуванні

Цей модуль забезпечує динамічне керування відображенням заголовка сторінки залежно від дій користувача. При прокручуванні сторінки вниз заголовок автоматично приховується, звільняючи більше простору для контенту. При прокручуванні вгору або при поверненні у верхню частину сторінки — заголовок знову з’являється.

JavaScript-логіка

Модуль підключається як окрема функція initScrollControlledHeader, яку необхідно викликати під час ініціалізації застосунку. В основі реалізації — слухач події scroll, який:

  • додає клас scroll-header, коли сторінка прокручена більше ніж на 10 пікселів (можна використати для зміни стилю заголовка після скролу);
  • змінює значення top у стилі заголовка для його приховування або показу залежно від напрямку прокрутки.

  let lastScrollTop = 0;
  const header = document.querySelector('.header');
  window.addEventListener('scroll', () => {
    const scrollTop = window.pageYOffset;
    header.classList.toggle('scroll-header', scrollTop > 10);
    header.style.top = (scrollTop <= 10 || scrollTop < lastScrollTop) ? '0' : '-100px';
    lastScrollTop = scrollTop;
  });

Необхідні стилі

Щоб скрипт працював коректно, для заголовка необхідно задати такі стилі:


.header {
  z-index: 100;
  width: 100%;
  position: fixed;
  top: 0;
  left: 0;
  transition: top 0.3s ease;
}

Рекомендації

  • Змінюйте значення -100px у стилі top, якщо висота вашого заголовка відрізняється.
  • Перевірте адаптивність поведінки заголовка на різних пристроях.
  • За потреби додайте throttle/debounce для оптимізації частоти викликів scroll-події.
