Скрипт приховування/показу заголовка при прокручуванні

У цьому розділі описано функціонал, який забезпечує динамічне приховування та показ заголовка сторінки (header) залежно від напряму прокручування сторінки.

Призначення

Скрипт додає інтерактивність до заголовка, приховуючи його під час прокручування сторінки вниз і показуючи при прокручуванні вгору або при поверненні до початку сторінки.

Реалізація

Логіка реалізована за допомогою обробника події scroll. Скрипт знаходиться в окремому файлі, scrollHeader.js, який підключається через app.js. 

          
      let lastScrollTop = 0;
      window.addEventListener('scroll', () => {
          const scrollTop = window.pageYOffset;
          document.querySelector('.header').style.top = (scrollTop <= 10 || scrollTop < lastScrollTop) ? '0' : '-100px';
          lastScrollTop = scrollTop;
      });

Структура проекту

Файли, пов'язані з функціоналом, мають наступну структуру: 

          
      project-folder/
      ├── src/
      │   ├── js/
      │   │   ├── functions/
      │   │   │   └── scrollHeader.js
      │   │   └── app.js

Стилі заголовка

Для коректної роботи функціоналу необхідно застосувати наступні стилі до заголовка: 

          
      .header {
          z-index: 100;
          width: 100%;
          position: fixed;
          top: 0;
          left: 0;
          transition: top 0.3s ease;
      }

Рекомендації

Рекомендується використовувати властивість transition для забезпечення плавного переміщення заголовка при зміні його позиції.
